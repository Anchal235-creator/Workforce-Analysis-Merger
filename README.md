 Workforce Analysis - Merger Case Study

Problem Statement
This project is based on a real world case study involving a merger between Technova and DataVerse. The objective is to optimize workforce structure while ensuring fairness, cost reduction, and improved productivity.

Objectives
- Reduce workforce cost by at least 10%
- Maximize output efficiency
- Maintain fair employee representation
- Retain high performing employees (A rated)
- Ensure cultural balance and employee retention

 Approach
- Analyzed employee dataset (performance, salary, tenure)
- Categorized employees into A, B, C ratings
- Applied constraints for cost reduction and fairness
- Optimized workforce distribution across departments

Tools Used
- Microsoft Excel

Files Included
- krishna.xlsx → Final workforce analysis solution
- Problem_statement.pdf → Case study details

 Outcome
Optimized workforce plan achieving better output to cost ratio while maintaining fairness and performance balance.
